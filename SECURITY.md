# Security Policy

## Reporting a Vulnerability

If you believe you have found a security vulnerability, please report it
privately using GitHub Security Advisories for this repository.

- Go to the repository on GitHub.
- Click **Security** → **Advisories** → **Report a vulnerability**.
- Provide a clear description, steps to reproduce, and any relevant logs.

We will acknowledge receipt as soon as possible and work with you on a fix and
responsible disclosure.
